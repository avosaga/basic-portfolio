# Basic Portfolio

A simple project to calculate Portfolio profit based on stock information

## Requirements
NodeJS >= 16.x.x
<br>
Run `yarn install` or `npm install` before anything else

## Build
```
yarn build
```
or
```
npm run build
```

## Run tests
```
yarn test
```
or
```
npm run test
```

## Run linter
```
yarn lint
```
or
```
npm run lint
```

## Run example
```
yarn build && yarn start
```
or
```
npm run build && npm start
```
