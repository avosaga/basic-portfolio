# Basic Portfolio

A simple project to calculate Portfolio profit based on stock information

## Requirements
NodeJS >= 16.x.x

## Install and execute example
```
yarn install && yarn start
```
or
```
npm install && npm start
```

## Build
```
yarn build
```
or
```
npm run build
```

## Run tests
```
yarn test
```
or
```
npm run test
```

## Run linter
```
yarn lint
```
or
```
npm run lint
```
